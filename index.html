SecureTube

Extensão para o Chrome que procura por iframes com a url http://www.youtube.com, e substitui por https.

Download .zip Download .tar.gz View on GitHub

Vídeos incluídos em páginas, agora através de HTTPS!

Carregando a extensão.

  1. Visite chrome://extensions no seu navegador
  2. Certifique-se que a checkbox Modo desenvolvedor/Developer Mode ao lado direito está marcada
  3. Clique em Carregar extensão expandida…/Load unpacked extension…, isso abrirá uma tela de seleção de arquivos
  4. Selecione a pasta em que a extensão se encontra.

Voilá, o botão já deve aparecer no seu navegador.

Como usar

Simples. Apenas clique no ícone que se encontra ao lado direito da barra de endereço, todos os vídeos do youtube que estiverem na página passarão a usar HTTPS.

Desenvolvimento futuro

Alterar as URLs ao carregar a página, antes mesmo da página se mostrada pelo navegador.
